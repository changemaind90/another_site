# Skin Selection Page for CS:GO and Dota 2
For layout training only, not for commercial use
## Project Description 📝

- Free layout template for markup using Figma;
- Adaptive / responsive design;
- Later, I will migrate everything to React / Next.js;
- using AI a little bit.

## Tech Stack Used 🛠️

- HTML5 🌐
- CSS3 🎨
- JavaScript 📜

## Project Structure 📁

/fonts     // fonts
/gallery   // folder with skins images  
/scripts   // scripts  
/styles    // styles CSS-only 
index.html // main page

## How to Run 🚀

1. Download or clone the repository 📥  
2. Open the `index.html` file in your browser 🌍

## How to Add New Skins ➕

- Place images into the `img/gallery` folder 🖼️  
- Update the array of skins in JavaScript by adding new objects with file paths and names:

```js
const skins = [
  { src: 'img/gallery/cs_skin1.png', name: 'CS:GO Skin 1' },
  { src: 'img/gallery/dota_skin1.png', name: 'Dota 2 Skin 1' },
  // add new skins here
];

- Бесплатный макет для верстки с Figma - страница выбора скинов CS:GO 2 | Dota 2 для покупки
- Adaptive/responsive дизайн;
- В дальнейшем переведу все на React\Nextjs;

## Используемый стек 🛠️

- HTML5 🌐

## Структура проекта 📁

/fonts     // шрифты

/scripts   // скрипты

1. Скачайте или склонируйте репозиторий 📥

## Как добавить новые скины ➕

- Поместите изображения в папку `img/gallery` 🖼️

```js

  { src: 'img/gallery/dota_skin1.png', name: 'Dota 2 Skin 1' },
  // добавляйте новые скины сюда
